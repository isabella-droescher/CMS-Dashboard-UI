# CMS Dashboard UI
Event-Management System mit CMS- und App-Interface für konsistente Nutzererlebnisse

## Überblick 
Dieses Projekt untersucht das Design und die konzeptionelle Frontend-Umsetzung eines verbundenen CMS- und App-Systems zur Verwaltung und Darstellung von Events.

## Problem
Event-Organisatoren benötigen eine effiziente Möglichkeit zur Erstellung und Verwaltung von Events, während Nutzer in der App eine klare und konsistente Darstellung erwarten.

## Ziel 
Entwicklung eines konsistenten CMS- und App-Systems mit klarer Struktur, intuitiver Navigation und einheitlichem Nutzererlebnis.

## Technologien / Tools 
Figma · UI/UX Design · Prototyping · User Flows

## Rolle
UI/UX Design · Prototyping · Informationsarchitektur

## Prozess
* Analyse der Informationsstruktur
* Erstellung von User Flows für CMS und App
* Entwicklung mehrerer Wireframe-Ansätze
* Ausarbeitung eines konsistenten UI-Konzepts in Figma

## Lösung
* CMS zur strukturierten Eventverwaltung
* App-Interface zur Darstellung und Entdeckung von Events
* Einheitliche visuelle und interaktive Sprache zwischen beiden Systemen
